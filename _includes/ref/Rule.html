Rule

Rule is one of the components of Plot, Link and Highlight

It creates a 

<rule>...</rule>

Rule can be added to any Plot, Link or Highlight via their method addRule {% highlight perl %} # Example: $t is your Plot instance, $r is your Rule instance $t->addRule($r){% endhighlight %}

Attributes

importance

INTEGER

condition

STRING

params

HASHREF

params is a hash that stores all key-value pairs. Only the keys and their values will be printed out in the respective blocks; params will not be printed.
